# Incident Scenario 2: Ransomware Activity

## Description

Unusual file encryption activity detected along with abnormal CPU spikes.

## Detection Method

Detected via:
- Suspicious process execution monitoring.
- Rapid file modification logs.

## Severity

Critical

## Response Steps

1. Isolate infected system from network.
2. Identify ransomware variant.
3. Restore from secure backup.
4. Patch vulnerabilities.
5. Conduct root cause analysis.

## Lessons Learned

- Maintain offline backups.
- Regular patch management.
